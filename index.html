IoT Air Quality & Environmental Monitoring Device

Project Overview

This wireless IoT device is built around an ESP32 microcontroller. It periodically collects air quality and environmental data from multiple sensors, formats the readings into a JSON payload, and sends it to a cloud API endpoint every 10 minutes over Wi-Fi.

Data was collected continuously for 2 days. The charts below represent only a small portion of the dataset for visualization purposes. They are not real-time.

System Overview

  • BME280: Temperature, Humidity, Pressure, Altitude
  • SCD30: CO₂ concentration
  • SPS30: PM1.0, PM2.5, PM4.0, PM10, NC0.5, NC1.0, NC2.5, NC4.0, NC10, Typical particle size
  • SGP40: VOC raw values, TVOC index

Key Features

  • Periodic data collection every 10 minutes
  • Multi-sensor integration into a single JSON payload
  • Reliable Wi-Fi communication with auto reconnect
  • Data validation and error handling for accurate readings

Technology Stack

  • Hardware: ESP32, BME280, SCD30, SPS30, SGP40
  • Software: Arduino IDE (C++), JSON formatting
  • Communication: HTTP POST over Wi-Fi to cloud API endpoint

Outcome

This device provides a reliable and autonomous solution for monitoring air quality and environmental conditions. The charts display a small portion of the 2-day dataset to illustrate trends in sensor outputs.
