IoT Air Quality & Environmental Monitoring Device

Project Overview

This project is a wireless IoT device built with an ESP32 microcontroller that collects air quality and environmental data from multiple sensors and sends it to a cloud API endpoint over Wi-Fi every 10 minutes.

Data was collected continuously for 2 days, and the charts shown here only display a small portion of the dataset for demonstration and understanding purposes.

System Overview

  • BME280: Measures Temperature, Humidity, Pressure, and Altitude
  • SCD30: Measures CO₂ concentration
  • SPS30: Measures Particulate Matter (PM1.0–PM10) and particle counts
  • SGP40: Measures VOC raw values and computes TVOC index

Key Features

  • Periodic Data Collection: Captures readings from multiple sensors every 10 minutes
  • Multi-Sensor Integration: Combines environmental and air quality measurements into a single JSON payload
  • Wi-Fi Communication: Sends data to a cloud API endpoint and reconnects automatically if disconnected
  • Data Validation: Flags invalid sensor readings

Technology Stack

  • Hardware: ESP32, BME280, SCD30, SPS30, SGP40
  • Software: Arduino IDE (C++), JSON formatting
  • Communication: HTTP POST over Wi-Fi

Outcome

The device provides a reliable and autonomous solution for monitoring air quality and environmental conditions. The charts presented are illustrative, showing only a small part of the 2-day data collection to help understand trends.
