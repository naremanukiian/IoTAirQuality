IoT Air Quality & Environmental Monitoring Device

Project Overview

This project is a wireless IoT device built with an ESP32 microcontroller that periodically collects air quality and environmental data from multiple sensors and sends it to a cloud API endpoint over Wi-Fi. The device operates autonomously, performing sensor initialization, data reading, validation, JSON formatting, and transmission every 10 minutes.

System Overview

  • BME280 – Measures Temperature (°C), Humidity (%), Pressure (hPa), and Altitude (m).
  • SCD30 – Measures CO₂ concentration (ppm).
  • SPS30 – Measures Particulate Matter (PM1.0–PM10) and particle counts.
  • SGP40 – Measures VOC raw values and computes TVOC index.

Sensor readings are combined into JSON and sent via Wi-Fi every 10 minutes to a cloud API endpoint.

Key Features

  • Autonomous Multi-Sensor Monitoring
  • Reliable Wi-Fi Communication with Auto-Reconnect
  • Data Validation (SPS30 data verified via CRC)
  • Periodic Data Collection every 10 minutes
  • Structured JSON Payload Construction
