IoT Air Quality & Environmental Monitoring Device

Project Overview

This project is a wireless IoT device built with an ESP32 microcontroller that periodically measures air quality and environmental conditions, sending the data to the cloud for storage and analysis.

System Overview

The ESP32 device collects data from multiple sensors:

  • BME280 – temperature, humidity, pressure, and altitude
  • SCD30 – CO₂ concentration
  • SPS30 – Particulate matter (PM1.0–PM10) and particle counts
  • SGP40 – VOC raw measurement and TVOC index

Sensor readings are formatted in JSON and sent over Wi-Fi every 10 minutes to an AWS Lambda function, which timestamps the data (UTC+4) and stores it in S3 for centralized management.

Key Features

  • Periodic Wireless Data Collection – Monitors air quality and environmental conditions automatically.
  • Multi-Sensor Integration – Provides comprehensive environmental data from multiple sensor types.
  • Cloud Storage & Centralization – AWS Lambda and S3 allow long-term storage and easy access to historical data.
  • Reliable Operation – Includes CRC checks, Wi-Fi reconnection, and data validation for robustness.

Technology Stack

  • Hardware: ESP32, BME280, SCD30, SPS30, SGP40
  • Software: Arduino IDE (C++), Python (AWS Lambda), JSON
  • Cloud: AWS Lambda + S3
  • Communication: HTTP POST over Wi-Fi

Outcome

Delivers a reliable, periodic air quality and environmental monitoring solution for indoor or outdoor spaces, suitable for smart buildings, IoT analytics, and air quality research.
