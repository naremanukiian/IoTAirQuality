IoT Air Quality & Environmental Monitoring Device

Project Overview

This project is a wireless IoT device built using an ESP32 microcontroller that collected air quality and environmental data every 10 minutes from multiple sensors. The data was sent over Wi-Fi to a cloud API endpoint. In this dashboard, we visualize a subset of the historical data collected over two days.

System Overview

  • BME280: Temperature (°C), Humidity (%), Pressure (hPa), Altitude (m)
  • SCD30: CO₂ concentration (ppm)
  • SPS30: Particulate Matter (PM1.0–PM10) and particle counts
  • SGP40: VOC raw measurement and TVOC index

Key Features

  • Data collected every 10 minutes over a period of two days.
  • Multi-sensor monitoring integrated in a single ESP32 program.
  • Data validation with CRC for SPS30 and error handling for reliable operation.
  • Data transmitted in JSON format over Wi-Fi to a cloud API endpoint.
