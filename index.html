IoT Air Quality & Environmental Monitoring Device

Monitor your environment in real-time with easy-to-understand charts and actionable insights

Project Overview

This project focuses on developing a wireless IoT device using an ESP32 microcontroller to monitor air quality and environmental parameters in real-time. The device integrates multiple sensors to measure temperature, humidity, pressure, altitude, CO₂ levels, particulate matter, and VOC indexes. The collected data is sent every 10 minutes to a cloud API for storage and analysis.

For demonstration purposes, the charts below show a small sample of the two-day dataset. This allows users to visualize environmental trends and gain insights without overwhelming them with raw data.

System Overview

The system combines several sensors to capture a comprehensive view of air quality and environmental conditions:

  • BME280: Measures temperature, humidity, pressure, and altitude, providing fundamental environmental metrics.
  • SCD30: Measures CO₂ concentration in ppm, helping detect air quality and ventilation efficiency.
  • SPS30: Measures particulate matter from PM1.0 to PM10, tracking pollution levels in the air.
  • SGP40: Measures volatile organic compounds (VOC) and computes TVOC index for air quality monitoring.

Key Features

  • Periodic Data Collection: Captures accurate readings from multiple sensors every 10 minutes.
  • Multi-Sensor Integration: Combines all sensor readings into a single JSON payload for efficient processing.
  • Wi-Fi Communication: Sends data to a cloud API endpoint with automatic reconnection if the device is disconnected.
  • Data Validation: Checks sensor readings for inconsistencies, ensuring reliable information for analysis.
  • Visualization Ready: Data is structured to easily generate interactive charts and dashboards.

Technology Stack

This project leverages modern hardware and software tools to ensure accuracy, scalability, and easy maintenance:

  • Hardware: ESP32 microcontroller, BME280, SCD30, SPS30, SGP40 sensors
  • Software: Arduino IDE (C++), JSON formatting for structured data exchange
  • Communication: HTTP POST requests over Wi-Fi for seamless data transmission to cloud endpoints

Outcome

The device provides a reliable, autonomous solution for monitoring indoor and outdoor environmental conditions. Users can track trends in temperature, humidity, air pressure, altitude, CO₂ levels, and particulate matter. The charts demonstrate how even a small dataset can reveal patterns and help make informed decisions about ventilation, air purification, or other environmental adjustments.

This project serves as a foundation for building larger-scale environmental monitoring systems, integrating IoT data collection with cloud storage, analysis, and visualization tools.
