IoT Air Quality & Environmental Monitoring Device

A unified multi-sensor platform delivering accurate environmental and air-quality insights

Project Overview

This project was developed as part of the ClimateNet program, featuring an ESP32-based IoT device equipped with multiple environmental and air-quality sensors. My main contribution was integrating all the sensor modules (BME280, SCD30, SPS30, and SGP40) into a single operational workflow, ensuring synchronized data collection and successful API communication.

The device records sensor data every 10 minutes and packages more than 20 parameters into a structured JSON payload before sending it to a cloud API for storage. The charts below represent a small extracted portion from a two-day dataset and demonstrate how collected values can be visualized for analysis.

System Overview

The ESP32 collects the following measurements during each cycle:

  • Temperature (°C) – Ambient air temperature.
  • Humidity (%) – Relative moisture content.
  • Pressure (hPa) – Atmospheric pressure levels.
  • Altitude (m) – Calculated using pressure variations.
  • CO₂ (ppm) – Carbon dioxide concentration.
  • Particulate Matter (PM1.0, PM2.5, PM4.0, PM10) – Number of particles per cm³ per size class..
  • Particle Count (NC0.5, NC1.0, NC2.5, NC4.0, NC10) – Mass concentration of airborne particles across size ranges..
  • Typical Particle Size – Average detected particle diameter.
  • Raw VOC Signal (SGP40) – Gas sensor output.
  • TVOC Index – Total volatile organic compound index.
  • Timestamp – Exact measurement time.
  • Device ID – ESP32 identifier.

Combined, these readings create a complete snapshot of environmental changes and air-quality conditions for each recorded moment.

Key Features

  • Unified Sensor Integration: Four different sensors combined into a single operational system.
  • Structured JSON Output: Over 20 sensor parameters included per measurement.
  • Wi-Fi Auto-Recovery: Automatic reconnection ensures stable data delivery.
  • Comprehensive Air-Quality Insight: Includes CO₂, particulate matter, VOCs, humidity, and more.
  • Dashboard-Ready Data: Ideal for charts, analytics models, and real-time monitoring.

Technology Stack

The system is powered by robust IoT hardware and lightweight software:

  • Hardware: ESP32, BME280, SCD30, SPS30, SGP40
  • Software: Arduino C++, sensor libraries, JSON handling
  • Networking: Wi-Fi + HTTP POST to a cloud API
  • Data Format: Timestamped JSON entries with 20+ recorded fields

Outcome

This IoT setup monitors all major environmental indicators, from temperature and humidity to air pressure, altitude, particles, CO₂, and VOC levels.

The collected information serves as a solid starting point for creating dashboards, detecting trends, setting up alerts, and expanding into real-time monitoring.
