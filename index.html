IoT Air Quality & Environmental Monitoring Device

A multi-sensor system that captures real environmental and air-quality data with clear visual insights

Project Overview

This project uses an ESP32-based IoT device to collect detailed environmental and air-quality measurements every 10 minutes. The device retrieves temperature, humidity, pressure, altitude, CO₂ concentration, particulate matter (PM1.0–PM10), particle count distributions, VOC levels, and raw gas sensor values. All measurements are packaged into a structured JSON payload and sent to a cloud API endpoint for storage and visualization.

The charts below display a small sample extracted from a two-day dataset. These charts are not real-time; they are shown to demonstrate how the collected data can be used to observe environmental patterns, trends, and anomalies.

System Overview

The ESP32 collects the following measurements on each cycle:

  • Temperature (°C) – Measures the ambient air temperature.
  • Humidity (%) – Indicates the moisture level in the air.
  • Pressure (hPa) – Atmospheric pressure measurement.
  • Altitude (m) – Estimated altitude calculated from pressure changes.
  • CO₂ (ppm) – Carbon dioxide concentration in parts per million.
  • Particulate Matter (PM1.0, PM2.5, PM4.0, PM10) – Levels of airborne particles of different sizes.
  • Particle Count (NC0.5, NC1.0, NC2.5, NC4.0, NC10) – Number of particles per cubic centimeter for each size range.
  • Typical Particle Size – Average detected particle diameter.
  • Raw VOC Value (SGP40) – Unprocessed gas sensor output representing volatile organic compounds.
  • TVOC Index – Simplified total VOC air-quality index (0–500).
  • Timestamp – The exact date and time of the measurement.
  • Device ID – Identifier for the ESP32 device sending the data.

Together, these metrics create a complete snapshot of both environmental conditions and air cleanliness at each measurement moment.

Key Features

  • Rich Environmental Data: More than 20 measurements collected every cycle.
  • Unified JSON Payload: All sensor values sent together in one structured package.
  • Automatic Wi-Fi Handling: The ESP32 reconnects automatically if Wi-Fi drops.
  • Accurate Air-Quality Insights: PM levels, CO₂ trends, VOC index, and particle distribution.
  • Visualization Ready: Data formatted for dashboards and analytics tools.

Technology Stack

The system is built using reliable IoT hardware and lightweight, efficient software:

  • Hardware: ESP32, BME280, SCD30, SPS30, SGP40
  • Software: Arduino C++, sensor libraries, JSON formatting
  • Networking: Wi-Fi + HTTP POST to a cloud API
  • Data: Timestamped JSON entries with 20+ parameters

Outcome

This project delivers a complete environmental monitoring solution capable of analyzing temperature, humidity, pressure, altitude, CO₂ buildup, particulate pollution, VOC levels, and particle size distributions. Even a short dataset provides meaningful insights about indoor or outdoor air quality.

The system serves as a strong foundation for advanced dashboards, prediction models, and real-time IoT monitoring platforms with alerts, mapping, and long-term analytics.
