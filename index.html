Project Overview

This system uses an ESP32-based IoT device to record high-resolution environmental and air-quality data at fixed 10-minute intervals. Each cycle captures temperature, humidity, pressure, altitude, CO₂ concentration, particulate matter levels, particle counts, VOC readings, and raw gas-sensor values. All measurements are combined into a structured JSON payload and transmitted to a cloud API endpoint for storage and visualization.

The visual charts below show a small sample extracted from a two-day dataset. They demonstrate how the collected data can be interpreted to identify trends, variations, and general environmental behavior.

System Overview

The ESP32 collects the following parameters on each measurement cycle:

  • Temperature (°C) – Ambient air temperature used to assess thermal conditions.
  • Humidity (%) – Relative moisture level in the environment.
  • Pressure (hPa) – Atmospheric pressure reading used for altitude estimation.
  • Altitude (m) – Calculated altitude derived from pressure changes.
  • CO₂ (ppm) – Carbon dioxide concentration, a key indoor-air quality indicator.
  • Particulate Matter: PM1.0, PM2.5, PM4.0, PM10 – Mass concentration of airborne particles across size ranges.
  • Particle Count (NC): NC0.5, NC1.0, NC2.5, NC4.0, NC10 – Number of particles per cm³ per size class.
  • Typical Particle Size – Average particle diameter detected.
  • Raw VOC Value (SGP40) – Unprocessed gas-sensor output before index conversion.
  • TVOC Index (0–500) – Processed air-quality index representing total VOC levels.
  • Timestamp – Exact moment the measurement was recorded.
  • Device ID – Unique identifier of the ESP32 device.

Together, these values form a complete environmental snapshot, enabling detailed monitoring of atmospheric and air-quality conditions.

Key Features

  • Comprehensive Sensor Suite: Over 20 parameters recorded during every cycle.
  • Structured JSON Format: All readings packaged into a single consistent payload.
  • Reliable Connectivity: Automatic Wi-Fi recovery ensures stable data delivery.
  • Air-Quality Intelligence: Includes particle concentration, CO₂ trends, VOC indicators, and size distributions.
  • Dashboard-Ready Output: Timestamped data suitable for visualization tools.

Technology Stack

  • Hardware: ESP32, BME280, SCD30, SPS30, SGP40
  • Software: Arduino C++ with sensor-library integrations
  • Connectivity: Wi-Fi + HTTP POST for cloud communication
  • Data Format: Time-indexed JSON entries containing all measurement values

Outcome

The system provides a complete environmental monitoring solution capable of tracking temperature, humidity, pressure, CO₂ buildup, particulate pollution, and VOC behavior. Even small datasets deliver meaningful insights, supporting dashboards, time-series analysis, predictive modeling, and long-term air-quality monitoring.
