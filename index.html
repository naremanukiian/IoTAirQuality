IoT Air Quality & Environmental Monitoring Device

Monitor your environment in real-time with easy-to-understand charts

Project Overview

This wireless IoT device uses an ESP32 microcontroller to collect air quality and environmental data from multiple sensors and sends it to a cloud endpoint every 10 minutes.

The charts below display a sample of 2-day data for visualization purposes.

System Overview

  • BME280: Measures Temperature, Humidity, Pressure, and Altitude
  • SCD30: Measures CO₂ concentration
  • SPS30: Measures Particulate Matter (PM1.0–PM10)
  • SGP40: Measures VOC raw values and computes TVOC index

Key Features

  • Periodic data collection every 10 minutes
  • Multi-sensor integration into a single JSON payload
  • Wi-Fi communication with auto reconnect
  • Data validation for accurate readings

Technology Stack

  • Hardware: ESP32, BME280, SCD30, SPS30, SGP40
  • Software: Arduino IDE (C++), JSON formatting
  • Communication: HTTP POST over Wi-Fi

Outcome

The device provides reliable monitoring of air quality and environmental conditions. Charts show a small sample to help visualize trends.
